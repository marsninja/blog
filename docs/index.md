## Quick Example

Here's a simple Jac program you can run right here:

<div class="code-block">
```jac
with entry {
    print("Hello from Jac!");

    for i in range(5) {
        print(f"Count: {i}");
    }
}
```
</div>

Click the "Run" button above to execute the code!
